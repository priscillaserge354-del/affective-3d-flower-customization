# WeChat Garden: 2025 全球青年情绪数字化孪生花园
**项目性质**：数字化艺术、人机交互与情感心理学交叉研究实验项目
**开发周期**：2025.06 - 2025.12 (全阶段开发迭代期)
**协作团队**：李海芳、尹玺权、徐成果、戴奕涵、宋欣

---

## 1. 项目愿景 (Product Vision)
WeChat Garden 致力于为当代大学生提供一个非线性的、具象化的情感存储空间。在信息碎片化的时代，传统的文字日记难以承载复杂的情绪波动。本项目通过 WebGL 3D 渲染技术，将 365 天的数字化脚印转化为一朵朵在赛博空间中随风摇曳、具有生命力的花朵，构建一个属于用户个人的“情绪数字孪生花园”。

## 2. 核心演示功能 (Core Features)

### 2.1 跨模态情感融合录入 (Cross-modal Input)
系统不再局限于单一的文字记录，而是构建了三位一体的输入维度：
* **语义识别层 (NLP)**：深度解析文字背后的情感权值，精准捕捉细腻的心境。
* **声学特征提取 (Voice)**：模拟声纹波形分析，通过语速、音调的物理特征映射情绪波动。
* **视觉基调分析 (Image)**：提取上传照片的色彩分布与构图特征，作为情感判定的辅助因子。

### 2.2 3D 高性能可视化渲染 (Advanced Rendering)
* **日历矩阵架构**：基于 Three.js 开发，构建了 365 天全年度 3D 逻辑方阵。
* **实例化渲染技术 (Instanced Mesh)**：在保持 60FPS 满帧运行的前提下，实时渲染超过 3000 个独立的草丛与花卉个体，确保场景的宏大与精致。
* **动态光影系统**：自研风力摇摆着色器 (GLSL Shader)，赋予植物自然的物理动效，配合昼夜交替系统，实现沉浸式视觉体验。

### 2.3 大学生情景化智能算法 (Intelligent Analysis)
* **定制化语料库**：针对 Z 世代校园生活，手动构建了包含 500+ 高频词条的专用词库，涵盖“早八、查分、挂科、图书馆抢座、社团面试”等 60 余种真实生活情景。
* **6 维度情感空间**：支持开心、难过、疑问、日常、愤怒、焦虑六种情绪的精准映射。

## 3. 核心技术栈 (Technical Stack)
* **后端引擎**：Python 3.9 + Flask (RESTful API 架构)
* **图形学前端**：Three.js R128 + WebGL + Tailwind CSS
* **人工智能底层**：Google Gemini Pro 2025 (Generative AI 跨模态接口)
* **资产规范**：GLB 2.0 (Low-Poly 低多面体美学风格)

## 4. 快速部署指南 (Quick Start)
1.  **环境配置**：克隆项目至本地，确保 Python 环境就绪。执行 `pip install -r requirements.txt` 安装 Flask 与相关 AI 通信插件。
2.  **网络中继**：本项目 AI 分析功能依赖跨模态算法通信，演示时请确保本地代理通道（默认 2712 端口）畅通。
3.  **核心启动**：在终端运行 `python app.py`。
4.  **实时访问**：打开 Chrome 或 Edge 浏览器，访问 `http://127.0.0.1:5000` 即可唤醒花园。
